Detailed Description

The FFAllocator, based on the ff_allocator.

Based on the ff_allocator, the FFAllocator might be used by any number of threads to dynamically allocate/deallocate memory.

ffa_wrapper* ff::FFAllocator::newAllocator ( size_t  max_size = 0,
int  _nslabs[N_SLABBUFFER] = 0,
bool  prealloc = true 
)
inline

Allocator factory method.

Each time this method is called it spawns a new memory allocator. The calling thread is registered as an allocator thread. Other threads willing to use this SlabCache have to register themselves to the shared leak queue of the allocator.

Parameters
max_sizePARAMETER NOT USED!
_nslabsarray specifying the allowed quantities of buffers. By default it is initialised to 0s
preallocflag that is used as a mask when creating a new slab. Default is true.
Returns
a ffa_wrapper object, that is in turn an extension of a object.
