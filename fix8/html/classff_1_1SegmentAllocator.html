void* ff::SegmentAllocator::newsegment ( size_t  segment_size)
inline

Allocates a new segment of (possibly aligned) memory.
This method uses standard malloc to allocate memory (or posix_memalign when possible). malloc should guarantee a sufficiently well aligned memory for any purpose, while posix_memalign is used to effectively allocate memory aligned in a defined way (i.e. the address of the allocated memory will be a multiple of a specific value).

Parameters
segment_sizethe quantity of memory to be allocated.
Returns
a pointer to the chunk of memory allocated.
