Detailed Description

template<typename CommImpl>
class ff::ff_dnode< CommImpl >

A class representing the distributed version of ff_node.

Defines the distributed version of FastFlow node.

A ff_dnode is actually a ff_node with an extra communication channel (external channel), which connects the edge-node of the graph with one or more edge-nodes of other FastFlow application graphs running on the same host or on different host(s).

It is implemented as a template: the template type CommImpl refers to the communication pattern that the programmer wishes to use to connect different ff_dnodes (i.e. unicast, broadcast, scatter, ondemand, fromAll, fromAny).
