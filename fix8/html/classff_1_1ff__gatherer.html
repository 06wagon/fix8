Detailed Description

A class representing the Collector node in a Farm skeleton.

This class models the gatherer, which wraps all the methods and structures used by the Collector node in a Farm skeleton. The farm can be seen as a three-stages pipeline, the stages being a ff_loadbalancer called emitter, a pool of ff_node called workers and - optionally - a ff_gatherer called collector. The Collector node can be used to gather the results coming from the computations executed by the pool of workers. The collector can also be connected to the emitter node via a feedback channel, in order to create a farm-with-feedback skeleton.
