typedef void*(* ff::ff_map::map_worker_F_t)(basePartitioner *const, int tid)

worker function type Function called by each worker thread as soon as an input task is received. The first parameter is the partitioner (that can be user-defined or one of those provided in the partitioners.hpp file) used the get a task partition for the worker. tid is the worker/thread id (from 0 to mapWorkers-1).
