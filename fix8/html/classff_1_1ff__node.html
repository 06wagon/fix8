Detailed Description

Describes the FastFlow node.

This class describes the ff_node, which is the basic building block of every skeleton in stream programming. It is used to encapsulate sequential portions of code implementing functions. ff_node defines 3 methods; two optional and one mandatory. The optional methods are (1) svc_init and (2) svc_end. The mandatory method is svc (pure virtual method). The svc_init method is called once at node initialization, while the svn_end method is called once when the end-of-stream (EOS) is received in input or when the svc method returns NULL. the svc method is called each time an input task is ready to be procedded.
