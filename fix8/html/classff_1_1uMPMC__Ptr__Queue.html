Detailed Description

An implementation of the unbounded Multi-Producer/Multi-Consumer queue.

This class implements an unbounded MPMC queue which does not require any special memory allocator to avoid dangling pointers. The implementation blends together the MPMC_Ptr_Queue and the uSWSR_Ptr_Buffer.

It uses internal atomic operations.
