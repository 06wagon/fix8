inline

It creates a new socket and pushes it into the active sockets list.

Parameters
Pis a flag to specify the role of the socket: if false, the new socket acts as a ROUTER (allows routing of messages to specific connections); if true the new socket acts as a DEALER (used for fair-queuing on input and for performing load-balancing on output toward a pool of collections).
Returns
