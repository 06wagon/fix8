This autogenerated documentation covers most of the classes, functions and global types used in a FIX8 project.

Some of the more commonly used classes include:

  • FIX8::Session encapsulates a FIX session. Your client or server session must inherit from this class.
  • FIX8::ClientSession a wrapper class for FIX session. Provides a simple client connection.
  • FIX8::ServerSession a wrapper class for FIX session. Provides a simple listening server.
  • FIX8::SessionInstance a wrapper class for FIX server session instance.
  • FIX8::BDBPersister is a BerkeleyDB backed FIX message persister.
  • FIX8::Configuration manages session configurations.
  • FIX8::Connection provides a complete FIX connection.
  • FIX8::f8Exception the Fix8 exceptions.
  • FIX8::Field manages and provides access to FIX data fields.
  • FIX8::FileLogger is an asynchronous logger.
  • FIX8::Message is a complete Fix message with header, body and trailer.
  • FIX8::Thread is a modern pthread wrapper.
  • FIX8::SessionID a Quickfix style session id.
