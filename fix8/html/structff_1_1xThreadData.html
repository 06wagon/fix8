Detailed Description

A buffer shared among threads (i.e. leak queue)

A buffer that is used as a channel for exchanging shared data among threads. It is built upon FastFlow's unbounded Single-Writer / Single-Reader queues (uSWSR_Ptr_Buffer). These buffers guarantee deadlock avoidance and scalability in stream-oriented applications.
