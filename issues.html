Found a problem ? Check the support group first as there may be help available already. If you think you've uncovered an issue then please report it so we can take a look.




To make a report, create an account on our issues page and select Create Issue. Please make sure you provide as much detail as you can; how to reproduce the problem; any supporting files that help. The more detail the better.
